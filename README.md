Zen Finance Tracker

Zen Finance Tracker is a modern personal finance tracking web application built using Node.js, Express, HTML, CSS, and JavaScript.

Users can:

Add income and expenses
Track financial balance
View analytics charts
Manage transactions in a clean UI
Run the app locally or deploy online
